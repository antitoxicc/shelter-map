# Shelter Map

Статический фронтенд для карты укрытий на `Leaflet` + `Supabase`.

## Что уже есть

- автоматическое определение позиции при заходе;
- кнопка обновления геопозиции;
- показ 3-5 ближайших подтверждённых точек с расстоянием;
- карточки точек с описанием и переходом в Google Maps;
- разные маркеры для пользователя и укрытий;
- отправка новых точек пользователями в статус `pending`;
- админка для `approve / delete`.

## Как подключить Supabase

1. Создай проект в Supabase.
2. Выполни SQL из `supabase-schema.sql` в SQL Editor.
3. Открой `supabase-config.js` и подставь `SUPABASE_URL` и `SUPABASE_ANON_KEY`.
4. Создай администратора через Supabase Auth.
5. Добавь его `user_id` в таблицу `public.admin_users`.

## Файлы

- `index.html` — основная карта.
- `admin.html` — страница модерации.
- `app.js` — логика пользовательской карты.
- `admin.js` — логика админки.
- `styles.css` — общий UI.
- `supabase-schema.sql` — таблицы и RLS-политики.
